Código-fonte

O comando document.getElementById('teste') é escrito em linguagem JavaScript. Utilizamos a tag CODE para monoespaçar as letras.

Exemplo de código em linguagem Python.

        
    num = int(input('Digite um número'))
    if num % 2 == 0:
    print(f'O número {num} é PAR')
    else:
    print(f'O número {num} é ÍMPAR')
    print('Fim do programa')

Somente a tag CODE não é suficiente para deixar o código pré-formatado. Precisamos envelopar na tag PRE. Aí sim o código ficará indentado.
Obs.: Se quisermos retirar a indentação, basta selecionar o código, sem as tags, apertar Shift + Tab. Para aumentar a indentação novamente, basta selecionar o código e apertar Tab somente.

Segundo Jeff Noble, no seu livro HTML para Leigos:

A diferença entre elementos inline e um bloco de texto é importante. Os elementos HTML neste capítulo descrevem os blocos de texto. Um elemento inline é uma palavra ou grupo de palavras dentro de um elemento do bloco (por exemplo, elementos com ênfase no texto, tais como <em> ou <strong>.)

A tag utilizada é a blockquote.
